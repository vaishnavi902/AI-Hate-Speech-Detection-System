# 🛡️ HateShield - AI Hate Speech Detection System

## 📌 Project Overview
HateShield is an AI-powered web application designed to detect hate speech in user-provided text. The system uses machine learning techniques to classify content and provides real-time results, analytics, and chatbot-based explanations.

---

## 🚀 Features
- 🔍 Detects hate speech in text input
- 🤖 AI-based classification using NLP
- 📊 Dashboard with analytics (charts)
- 💬 Chatbot for user interaction and explanation
- 🗄️ Stores prediction history in database
- ☁️ Cloud deployment (Render)
